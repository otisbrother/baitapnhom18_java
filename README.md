## Yêu Cầu Môi Trường
- **JDK 17**
- **Trình biên dịch:** `javac`
- **IDE khuyến nghị:** VS Code, IntelliJ IDEA, Eclipse

## Hướng Dẫn Chạy
1. **Biên dịch:** `javac src/*.java`
2. **Chuyển thư mục** `cd src`
3. **Chạy:** `java Main.java`
